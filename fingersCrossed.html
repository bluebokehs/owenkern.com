Project Details

Background

We decided to create a tabletop game that combined the basic traversal of linear boards with the poker face antics in a game like Werewolf. The first challenge was to design an instruction manual that portayed the accurate tone.

Board Design

The next challenge was to create a board that aided the double crossing game mechanics of Werewolf, which is where the multiple paths come in.

Final Form

Here the final form of our tabletop game rendered out from Adobe Dimensions. We plan to crowdfund the development of this game assuming there are people interested.
